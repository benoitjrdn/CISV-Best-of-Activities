# CISV-Activities
This is a compilation of the best CISV activities. This is not an official resource and should not be treated as such.

This database is a work in progress and can be updated any time.
Latest update : 2026/03/31

2. About
3. How to use it

### Purpose

Organizing great activities is one of the key aspects of being a great CISV Leader.
However, the gamebooks are old, not always updated, and not always easy to use.
This repo attempts to solve those issues by creating an intuitive and dynamic database.

### About

I made this first for myself, to easily find the best-suited activities for the camps I organized.
Most activities are a compilation of the best ones that I found in all the gamebooks.
Some are activities my friends have organized or came up during a camp, and some are the ones that I created myself.

Some of my dear CISV friends prompted me to publish it, in order to have the most amount of people benefit from it.

Please understand that I tried to make it comprehensible, but some activities might be explained in french and not translated yet. Some others are very briefly explained, and some others are still in the process of being made.

Updates will be made.
You can email me at benoit.jrdan@gmail.com for any inquiries. Feel free to send an activity as well, if you want me to publish it !

### How to use it

The files are text files (markdown). They can be downloaded and read as is.

However, this is far from the best way of doing it. I will succinctly explain how to use it as a database you can filter.

1. Download the files anywhere you want - I suggest Documents/CISV
2. Download Obisidan on www.obsidian.md
3. Open Obsidian and select "Open Folder as Vault" and select the file path of Document/CISV (or wherever you downloaded the files).
4. Open the "CISV Activities.base" file
