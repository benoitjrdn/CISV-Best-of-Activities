# Chairs Monster

### Goal

Cooperate and learn strategic thinking.
Have fun.

### Description

A monster coming from another dimension will enter your reality if he is able to sit down on a chair.
The kids have to prevent him from sitting down : but only one can stand at a time.

### Instructions

##### Setting up the game

- take one chair for every participant, plus one
- everybody sits on a chair (one should be free)
- set them apart randomly (best outside on a big field), a few meters apart

##### Game rules

- the monster (leader at first) walks (slowly ! not allowed to run !) towards the chair
- ONE kid stands up and runs to the chair, so you can't sit on it
	- by doing that, he frees up his chair
	- you walk to this new, empty chair
	- another kid stands up and run
- if two or more kids get up, the group loses (be more or less tolerant depending on the age)
- The kids win if they last 1 / 2 / 3 minutes (whatever you want).

> [!tip] 
> - Of course the best strategy is to communicate, else everybody gets up
> - It makes more sense to get up and run to the chair if you are far from it, so that the monster has to walk a lot
> - organize a break for discussion and elaborating a strategy because the kids will not be smart about it at first
> 	- plan multiple breaks actually

##### Increasing the difficulty

- not allowed to talk proper words (/at all for older groups)
- the faster the monster the more difficult
- you can only get up once (the group wins if everybody went once)
