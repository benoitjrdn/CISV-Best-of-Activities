# Bulldog
épervier

### Goal

Have fun, run around

### Description

The kids need to go from `A` to `B` back and forth repeatedly.
One (or two) person is in the middle trying to catch them.

#### Alternative Versions

The people who were caught can either help and run or have to stand in the same spot they were caught.
Ask the kids for variations, this is a super common game they might know some fun stuff.
