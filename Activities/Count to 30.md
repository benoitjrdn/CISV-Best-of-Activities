# Count to 30

### Goal

Learn about cooperation

### Description

The kids have to count to [30]. They are not allowed to talk when the round starts. If they really struggle, you can let them discuss a strategy.
Only one can say a number at a time.
If two people say it at the same time, they have to start over.

Can be done after unsuccessful cooperation games for instance.
