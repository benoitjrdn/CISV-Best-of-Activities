# Build a City

### Goal

Learn about different political regimes

### Description

> [!todo] Materials
> - [ ] Something to build cities


Four groups, unaware of the regimes of the other groups, have to build a city.

| Group   | Building style                         |
| ------- | -------------------------------------- |
| Group 1 | Majority decides what and how to build |
| Group 2 | Only one decides                       |
| Group 3 | Everybody has to agree                 |
| Group 4 | Everyone for themself                  |
