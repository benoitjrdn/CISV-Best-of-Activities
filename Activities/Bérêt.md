# Bérêt

### Goal

Have fun, run around

### Description

Two teams. 
Each player gets a number (version 1). Or one gets a number and one gets a letter (version 2).

Each team makes a row, facing each other, 10-20 meters apart.
Put a hat / shoe / whatever in the middle.

One Organisator calls a random number (/number x letter) each round.
When called, they have to run to the middle, grab a hat or whatever you put there, and run back without the other catching them.

### Instructions

Groups of ~10 is best.
