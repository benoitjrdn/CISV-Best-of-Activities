# Give this to...

### Goal

Learn names, have fun

### Description

The leader of the game starts by giving an object to someone and saying “Give this to ... “: The recipient then gives the object to the person named saying “I got this from ... . Now give it to ..” More and more objects are added into the game, so everyone is giving and receiving all the time. Later it is fun to pass larger objects like chairs and people.
