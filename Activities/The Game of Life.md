# The Game of Life

Activity by Lucas, CISV France

### Goal

Realize how economic inequalities affect where we end up in life.

### Description

This activity simulates life.
Some people start with more of a currency.
There are multiple stations, each being fun (a movie, a massage center, an arts and crafts room, lullabies, pool...).
Accessing them requires a certain level of education and money.
You can work jobs to win money, to pay for a degree, where you can earn more.
