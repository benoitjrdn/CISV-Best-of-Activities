# Ecosystem Running Game

### Goal

Learn what an ecosystem is, and why they can be fragile.

### Description

Un jeu ou les enfants sont :

- des plantes (80%)
- des lapins (15%)
- des renards (5%)

Le plantes ne bougent pas et ont un stock de papier.
Les Leaders leur en donnent toutes les 5 minutes un nouveau stock.
Si il est épuisé, la plante meurt.

Les lapins vont voir les plantes qui leur donnent un papier.
Ils doivent faire des tours et ne pas demander à la même plante plusieurs fois d'affilé.


Les enfants doivent donner une feuille
