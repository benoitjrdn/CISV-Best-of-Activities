# Triage Game

### Goal

Break the ice

### Description

Tell the kids that they have to get in order without speaking.

> [!list] Suggestions
> - [ ] Sort by age
> - [ ] Sort by birthday (January 1st -> December 31)
> - [ ] (Sort by height) (be mindful, height can be an insecurity, especially for boys)
> - [ ] Sort by name in alphabetical order
> - [ ] Sort by the name of your country
> - [ ] ...
