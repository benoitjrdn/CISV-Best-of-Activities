# Save the Reefs

Je veux dire les dunes plutot
expliquer en quoi elles stoppent les grosses vagues

> [!wip] 

### Goal

Learn about how nature does a great job at preventing degradation of the *dunes*, and why destroying it like we do actually is bad for everyone.

### Description

In a terrarium, simulate waves on one end. There is land on the other, with houses.
They have to create the most efficient way possible to preserve the dunes.

Then show them how the best way is actually nature's way.
